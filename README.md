# CyberPravo Telegram Bot

Telegram-бот для конкурса **#КиберПраво** на Python + aiogram.

## Функции
- Проверка ссылок на признаки фишинга
- Генератор сложных паролей
- Квиз по цифровой гигиене и ответственности
- Мини-база знаний

## Установка и запуск
1. Создайте виртуальное окружение:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте `.env` в корне проекта:
   ```env
   BOT_TOKEN=<ваш токен бота>
   ```
   Можно также скопировать шаблон:
   ```bash
   cp .env.example .env
   ```
4. Запуск:
   ```bash
   python main.py
   ```

## Почему раньше возникала ошибка BOT_TOKEN
Если `BOT_TOKEN` не задан, бот завершится с понятным сообщением.
Теперь `main.py` автоматически читает `.env` (без дополнительных библиотек),
поэтому достаточно создать файл `.env` рядом с `main.py`.

## Примечание
- Логика фишинг-детектора демонстрационная; для продакшена подключайте внешние reputation API.
